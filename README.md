# React Vite Fullstack App

A modern web application built with React and Vite, served by an Express backend.

## Features
- **Frontend**: React 18 with Vite for fast HMR and bundling.
- **Backend**: Express server integrated via `vite-express`.
- **Development**: Concurrent frontend/backend development.

## Tech Stack
- React
- Vite
- Express
- Node.js

## Setup
1. `npm install`
2. `npm run dev` (Starts server with nodemon)
